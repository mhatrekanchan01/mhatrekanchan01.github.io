How I Choose Tools

Useful beats impressive.

I prefer tools that HR can maintain. A workflow is only valuable if it reduces friction without creating a system that is too complex to update.

I look for places where a small system can make work more consistent: reminders, trackers, first-draft SOPs, dashboards, message templates, and follow-up rhythms that keep people work moving.
